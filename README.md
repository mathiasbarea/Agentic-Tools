## 🚀 Installation

Installing these skills is straightforward. Simply provide your OpenClaw agent with the skill's GitHub URL and ask it to install it for you. 

Depending on your needs, you can choose between two installation scopes:

* **Global Installation:** Ask the agent to install it in the `.openclaw/skills` directory if you want the skill to be available to all your agents.
* **Workspace Installation:** Ask the agent to install it in your `<workspace>/skills` directory if you want the skill to be restricted to a specific agent.
